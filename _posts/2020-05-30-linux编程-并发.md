## 常用函数

### kill

send signal to a process

```c
#include <sys/types.h>
#include <signal.h>
int kill(pid_t pid, int sig);
```

### raise

send a signal to the caller

```c
#include <signal.h>
int raise(int sig);
```

### alarm

set an alarm clock for delivery of a signal

```c
#include <unistd.h>
unsigned int alarm(unsigned int seconds);
```

### pause

wait for signal

```c
#include <unistd.h>
int pause(void);
```
